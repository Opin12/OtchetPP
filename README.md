5.5 Задания для выполнения в организации (обязательный перечень для каждого студента)
Выполняются на реальном оборудовании в организации, где проходит практика.
По каждому пункту требуется прикрепить фото/скриншоты на доску YouGile.
Подготовка оборудования на площадке практики – диагностика ПК, сборка, подключение периферии, проверка работоспособности.
Инсталляция и настройка ПО в реальных условиях – установка РЕД ОС, настройка политик безопасности, пользователей, обновлений.
Настройка корпоративного подключения к Интернету – анализ требований организации, выбор оптимальной технологии и тарифа, настройка маршрутизатора.
Установка специализированного ПО и драйверов – установка ПО для сканеров, принтеров, систем документооборота; настройка клиентских приложений.
Учёт и управление сетевым трафиком – настройка системы сбора статистики по пользователям, формирование отчётов для ИБ-службы.
Интеграция локальной сети организации в корпоративную ИТ-инфраструктуру – настройка VLAN, общих папок, доменной интеграции (при наличии), доступа к сетевым ресурсам.
Настройка серверного ПО для системы контроля и управления доступом (СКУД) – установка и настройка сервера СКУД (на базе ARMA или ViPNet), создание базы пользователей, настройка прав доступа. Фото интерфейса или конфигураций обязательно.
GitHub: фото и краткие отчёты → Практика/Отчёты/; конфигурации сервера СКУД → Практика/СКУД_сервер/.
